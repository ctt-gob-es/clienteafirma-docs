Version 1.6
-----------
- Se corrige un cierre inesperado de la aplicación cuando en la configuración de firma proporcionada desde el navegador web no se envían los datos a firmar y, aún así, es demasiado grande para enviarse sin servidor intermedio.
- Se corrige la generación de firmas PAdES visibles.
- Se agrega el soporte para la operación de selección de certificados al invocarse desde un navegador web.

Version 1.3
-----------
- Soporte para la firma de ficheros de mayor tamaño.

- Corrección de errores y mejora en la estabilidad.


Version 1.2
-----------
- Mejor adhesión a estándar de formatos de firma: no se incluye la descripción del contenido en contrafirmas CAdES, se permite configurar el atributo commitmentTypeIndication y se deja de poder configurar el atributo certifiedRole (debería estar firmado por una tercera entidad).
- Se agrega soporte a la configuración "AUTO" como formato. Detección del formato de las firmas proporcionadas en cofirma y contrafirma.
- Soporte multi-idioma mejorado.
- Corrección de errores y mejora en la estabilidad.
